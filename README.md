## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. **Upload Images**: Drag and drop images or click to select files
2. **Configure Options**: Set resize dimensions, format, and compression level
3. **Compress**: Click "Compress Images" to process
4. **Preview**: View side-by-side comparison with size statistics
5. **Download**: Download individual images or all as ZIP
